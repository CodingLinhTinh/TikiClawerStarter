## ⚙️ Hướng dẫn sử dụng
<b>Sử dụng Python để chạy</b>
- Gõ lệnh cài đặt biến môi trường
```python -m venv env
```
- Activate biến môi trường:
```env\Scripts\activate
```
- Chạy file requirements.txt 
```pip install -r requirements.txt 
```
- Chạy file main.py
```python main.py
```
